# تفعيل Prerendering عبر Netlify المدمج

بما أن حساب Prerender.io المجاني قد وصل للحد الأقصى، فإن أفضل حل هو استخدام ميزة نيتلفاي المدمجة.

## الخطوات المطلوبة:

### 1. تفعيل الخيار في لوحة تحكم Netlify
1. اذهب إلى **Site settings** في لوحة تحكم نيتلفاي.
2. اختر **Build & deploy** من القائمة الجانبية.
3. ابحث عن قسم **Prerendering**.
4. اضغط على زر **Edit settings** ثم اختر **Enable**.
5. اضغط **Save**.

### 2. نشر التغييرات (Push)
لقد قمت بحذف الأكواد السابقة التي قد تتعارض مع ميزة نيتلفاي المدمجة. يجب عليك الآن رفع هذه التغييرات:
```bash
git add .
git commit -m "Switch to Netlify built-in prerendering"
git push
```

## لماذا هذا الحل أفضل؟
- **مجاني**: مدمج مع استضافتك على نيتلفاي.
- **لا يحتاج توكن**: لن تظهر لك رسالة "Account paused" مرة أخرى.
- **تلقائي**: نيتلفاي سيتعامل مع جميع محركات البحث (Google, Facebook, Twitter) تلقائياً.

## كيفية التأكد من العمل
بعد تفعيل الخيار من لوحة التحكم وعمل Push، يمكنك الاختبار بنفس الأمر:
```powershell
curl.exe -A "Googlebot" "https://alsamah-store.com/" -I
```
ستلاحظ أن نيتلفاي يقوم الآن بمعالجة الطلب وتقديم نسخة مؤرشفة دون الحاجة لخدمات خارجية متعثرة.
