# 🎯 ابدأ من هنا

## ⚡ 3 خطوات فقط

### الخطوة 1: شغّل الأداة التفاعلية
```bash
npm run migrate:interactive
```

### الخطوة 2: أدخل البيانات المطلوبة
- رابط Supabase المصدر
- مفتاح الخدمة للمصدر
- رابط Supabase الهدف
- مفتاح الخدمة للهدف

### الخطوة 3: انتظر انتهاء النقل
```
✅ تم استخراج البيانات
✅ تم إدراج البيانات
✨ اكتمل النقل بنجاح!
```

---

## 📍 أين أحصل على المفاتيح؟

1. اذهب إلى [Supabase Dashboard](https://app.supabase.com)
2. اختر المشروع
3. اذهب إلى **Settings** → **API**
4. انسخ **Service Role Key**

---

## ✅ بعد انتهاء النقل

1. حدّث ملف `.env` بالمفاتيح الجديدة
2. أعد تشغيل الخادم: `npm run dev`
3. اختبر الموقع
