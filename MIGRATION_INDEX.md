# فهرس نقل قاعدة البيانات

دليل شامل لنقل محتوى قاعدة البيانات من Supabase إلى قاعدة بيانات أخرى.

## 📚 الأدلة الرئيسية

### 1. 🚀 [البدء السريع](./QUICK_MIGRATION.md)
**للمستخدمين المستعجلين**
- أسرع طريقة للبدء
- خطوات بسيطة وواضحة
- أمثلة عملية

### 2. 📖 [دليل النقل الشامل](./DATABASE_MIGRATION_GUIDE.md)
**للمستخدمين الذين يريدون التفاصيل**
- شرح مفصل لكل خطوة
- استكشاف الأخطاء
- خيارات متقدمة

### 3. 🔧 [إعداد قاعدة البيانات الهدف](./SETUP_TARGET_DATABASE.md)
**قبل البدء بالنقل**
- إنشاء مشروع Supabase جديد
- تطبيق الهجرات
- إعدادات الأمان

### 4. ✅ [التحقق من النقل](./VERIFY_MIGRATION.md)
**بعد انتهاء النقل**
- التحقق من البيانات
- اختبار الموقع
- قائمة التحقق النهائية

### 5. 🔄 [تبديل الموقع للاتصال بقاعدة جديدة](./SWITCH_DATABASE.md)
**تحديث الموقع**
- تحديث ملف `.env`
- إعادة تشغيل الخادم
- نشر التحديثات

### 6. 🖼️ [نقل الصور والملفات](./MIGRATE_IMAGES.md)
**نقل الملفات من Storage**
- نقل الصور
- تحديث الروابط
- سياسات الوصول

### 7. ↩️ [الرجوع للخلف](./ROLLBACK_MIGRATION.md)
**في حالة حدوث مشكلة**
- الرجوع السريع
- استعادة البيانات
- منع المشاكل

### 8. 📋 [الملخص](./DATABASE_MIGRATION_SUMMARY.md)
**نظرة عامة على كل شيء**
- الملفات المُنشأة
- الخطوات السريعة
- الأسئلة الشائعة

## 🎯 خطة النقل خطوة بخطوة

### المرحلة 1: التحضير
1. اقرأ [البدء السريع](./QUICK_MIGRATION.md)
2. اتبع [إعداد قاعدة البيانات الهدف](./SETUP_TARGET_DATABASE.md)
3. تأكد من تطبيق جميع الهجرات

### المرحلة 2: النقل
1. اختر طريقة النقل:
   - **التفاعلية:** `npm run migrate:interactive`
   - **باستخدام ملف البيئة:** `npm run migrate`
2. اتبع [دليل النقل الشامل](./DATABASE_MIGRATION_GUIDE.md) للتفاصيل

### المرحلة 3: التحقق
1. اتبع [التحقق من النقل](./VERIFY_MIGRATION.md)
2. تحقق من جميع البيانات
3. اختبر الموقع بشكل شامل

### المرحلة 4: التبديل
1. اتبع [تبديل الموقع](./SWITCH_DATABASE.md)
2. حدّث ملف `.env`
3. أعد تشغيل الخادم

### المرحلة 5: النشر
1. اختبر الموقع محلياً
2. نشّر التحديثات إلى الإنتاج
3. راقب الموقع للتأكد من عدم وجود مشاكل

## 🛠️ الأدوات المتاحة

### سكريبتات النقل

```bash
# الطريقة التفاعلية (الموصى به)
npm run migrate:interactive

# باستخدام ملف البيئة
npm run migrate
```

### ملفات التكوين

- `.env.migration.example` - مثال على ملف المتغيرات
- `scripts/migrate-database.js` - سكريبت النقل الأساسي
- `scripts/interactive-migrate.js` - أداة تفاعلية

## ❓ الأسئلة الشائعة

### س: من أين أبدأ؟
ج: اقرأ [البدء السريع](./QUICK_MIGRATION.md) أولاً.

### س: كم من الوقت يستغرق النقل؟
ج: عادة من ثوانٍ إلى دقائق حسب حجم البيانات.

### س: هل يمكن الرجوع للخلف؟
ج: نعم، اقرأ [الرجوع للخلف](./ROLLBACK_MIGRATION.md).

### س: ماذا لو حدث خطأ؟
ج: اقرأ استكشاف الأخطاء في الدليل المناسب.

### س: هل البيانات آمنة؟
ج: نعم، استخدم Service Role Keys وحافظ على سرية المفاتيح.
